## 🔌 MCP Basics

MCP = Model Context Protocol. Hiermee kan Claude praten met externe tools.

**Config locatie:** `~/.claude/settings.json`

**Structuur:**
```json
{
  "mcpServers": {
    "tool-naam": {
      "command": "npx",
      "args": ["-y", "@package/name"],
      "env": { "API_KEY": "jouw-key" }
    }
  }
}
```

**Populaire MCP servers:** filesystem, GitHub, Slack, Google Drive, Notion, Airtable
